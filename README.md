This project is a collection of three interactive web applications built using HTML, CSS, and JavaScript. 
It serves as a mini portfolio showcasing different frontend skills and API integrations.

---

## 🚀 Projects Included

### 🎬 Movie Search App
- Search for movies using an API
- Displays posters, ratings, and details
- Clean and responsive UI

### 🌐 Translator App
- Translate text between multiple languages
- Uses Google Translate API (unofficial endpoint)
- Features:
  - Custom dropdowns
  - Typing animation
  - Copy & clear functionality

### ☁️ Weather App
- Get real-time weather data by city
- Displays temperature, humidity, wind, etc.
- Modern dashboard-style UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Glassmorphism, animations)
- JavaScript (DOM manipulation, fetch API)
- Font Awesome Icons

---

## 🔗 API Documentation Links

- **Movie API**  
  https://www.themoviedb.org/documentation/api

- **Weather API**  
  https://openweathermap.org/api

- **Translator API (Unofficial)**  
  https://translate.googleapis.com/translate_a/single

---

## ⚙️ Key Implementation Details

- Custom UI components (replaced default `<select>` with custom dropdowns)
- Animated transitions (fade, typing effect, hover interactions)
- Responsive grid layout for project navigation
- Skeleton loader for async data fetching
- Dynamic API integration using `fetch()`
- Session-based UI behavior handling
